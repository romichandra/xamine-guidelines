# Xamine.ai Brand Guidelines

## 1. Brand Overview
**Mission**: Revolutionizing exam preparation with AI-powered personalization, rank prediction, and deep analytics for students and coaching institutes.
**Core Values**: Innovation, Precision, Accessibility, Excellence.

---

## 2. Logo
The Xamine.ai logo is the primary identifier of the brand.

### Variants
- **White**: Used on dark backgrounds and the Aurora background.
- **Black**: Used on light, solid backgrounds.

### Usage
- Maintain a minimum clear space equivalent to the height of the 'X'.
- Hover effect: Scale 1.05 with a spring transition.

---

## 3. Color Palette

### Primary Colors
| Name | Hex | Usage |
| :--- | :--- | :--- |
| **Primary Violet** | `#8B5CF6` | Main brand color, buttons, accents |
| **Deep Violet** | `#7C3AED` | Hover states, primary-dark |
| **Midnight Navy** | `#1a1130` | Dark mode background depth |

### Gradient Palette
- **Start**: `#a855f7` (Purple-500)
- **End**: `#6366f1` (Indigo-500)
- **Fluid Gradient**: Used for text and decorative elements.

### Grayscale (Neutral)
| Level | Hex | Usage |
| :--- | :--- | :--- |
| **White** | `#FFFFFF` | Light mode background, text |
| **Zinc-100** | `#F4F4F5` | Secondary backgrounds |
| **Zinc-500** | `#71717A` | Muted text, 3D wave ridges |
| **Zinc-900** | `#18181B` | Dark mode base background |

---

## 4. Typography

### Primary Font: **Geist** (Sans/Mono)
- **Sans**: Used for body text, UI labels, and technical data.
- **Mono**: Used for code snippets and specific data points.

### Display Font: **Plus Jakarta Sans**
- Used for headings and large display text.
- Characterized by high geometric clarity and modern feel.

### Hierarchy
- **H1**: Bold, 4xl-7xl, often with `.text-gradient`.
- **H2/H3**: Bold, 2xl-3xl, Slate-900 or White.
- **Body**: Regular, 16px, line-height 1.6.

---

## 5. Iconography
- **Library**: [Lucide React](https://lucide.dev/)
- **Style**: 1.5pt - 2.5pt stroke width, rounded ends.
- **Coloring**: Often used within tinted containers (`bg-accent/10`).

---

## 6. UI Design Patterns

### Glassmorphism
- **Class**: `.glass-card`
- **Properties**: White/Dark transparent background, 12px blur, subtle border.
- **Hover**: 1px translate up, increased shadow.

### Aurora Background
- **Effect**: 3D grayscale waves from a top-down view.
- **Animation**: Smooth 60s sliding motion with `mix-blend-difference`.

### Particle System
- Subtle floating dots in the background to provide depth.

---

## 7. Voice & Tone
- **Professional & Expert**: Clear, authoritative language about AI and analytics.
- **Tech-Forward**: Terms like "Ecosystem", "Personalization", "Deep Analytics".
- **Encouraging**: Call-to-actions like "Start for Free", "Join the Future".
